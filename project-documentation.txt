# Project Documentation: Headcanon Space

## 1. Feature Flow & Interaction Model

Headcanon Space is designed as a high-performance, instant-results creative suite for fandom writers and roleplayers. The feature flow across all tools follows a consistent, frictionless pattern.

### Core Interaction Loop
1.  **Selection**: Users navigate to a specific generator (Home, Character, Relationship, Scenario, or Random) based on their creative need.
2.  **Configuration**: 
    - **Home**: Input character name, fandom, tone (Soft, Dark, Chaotic, etc.), setting, and output focus.
    - **Character**: Focuses on a single character's traits, habits, and secrets.
    - **Relationship**: Takes two names and explores ship dynamics, rituals, and tension.
    - **Scenario**: Places characters in specific settings (War, School, Modern) with dramatic triggers.
    - **Random**: Zero-input mode for instant inspiration.
3.  **Generation (Engine Logic)**: 
    - The client-side engine (`generator.ts`) picks templates from curated pools based on user inputs.
    - It performs dynamic string replacement (e.g., swapping `{name}` with user input).
    - It applies "Tone Modifiers" and "Length Extensions" to ensure the output matches the desired vibe and depth.
4.  **Instant Feedback**: Results appear in seconds without page reloads, using smooth CSS animations (`animate-fade-up`, `animate-scale-in`).
5.  **Output Management**: 
    - **Copy**: One-click copy to clipboard.
    - **Regenerate**: Instantly swap for a new batch of ideas.
    - **Save**: Persists results to browser LocalStorage so they remain available after a refresh.

## 2. SEO Implementation Details

The project follows strict SEO best practices to ensure high visibility in search engines for "headcanon" and "writing prompt" related queries.

### Global Configuration (`layout.tsx`)
-   **Meta Title**: Free Headcanon Generator – Create Ideas Instantly
-   **Meta Description**: Generate creative headcanon ideas instantly with this free headcanon generator. Perfect for fandoms, characters, and stories.
-   **Canonical**: Points to `https://headcanonspace.com`.
-   **OpenGraph/Twitter**: Configured for rich social media previews.
-   **Favicon**: Custom SVG icon (`icon.svg`).

### Structured Data (JSON-LD)
We use complex Schema.org graphs to help Google understand the site's utility:
-   **SoftwareApplication**: Identifies the tool, its category (Utilities), and its free price.
-   **FAQPage**: Lists common questions on the home and blog pages to capture "People Also Ask" results.
-   **BreadcrumbList**: Clarifies site hierarchy.
-   **Article**: For blog posts to improve indexing in News/Discover.

### Heading Structure & Semantic HTML
-   **H1**: Each page has exactly one H1 focused on the primary keyword (e.g., "Character Headcanon Generator").
-   **H2/H3**: Used for feature descriptions, "How it Works" sections, and FAQs.
-   **Semantic Tags**: `<main>`, `<section>`, `<article>`, `<nav>`, and `<footer>` are used throughout.
